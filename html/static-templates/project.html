West Loop Residence

Chicago, IL / 2017

We were tasked with taking this urban concrete loft project from the 80's to present. Together with a wonderful client and a great team we were able to design this project in only 8 weeks, completing construction in 3.5 months. We created a custom millwork & case goods package that ties the entire home together, gutted the guest bathroom, revised the layout to make the larger guest bedroom more efficient and rehabbed the existing kitchen extensively using the existing cabinets. The flooring was carefully patched, repaired and stained matte grey to become a refreshing consistent element. The hanging lighting was added to again tie the entire home together.
